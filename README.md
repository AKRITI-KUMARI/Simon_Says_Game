<h1>Simon Says Game</h1>
<p>A fun, interactive web-based Simon Says game built using HTML, CSS, and JavaScript.</p>
<p>Test your memory and reflexes by repeating the sequence of colors correctly as the game progresses!</p>

 <h3>How to Play</h3>
- Press any key to start the game.
- Watch the sequence of colors highlighted.
- Repeat the sequence by clicking the correct colors in order.
- Each round adds a new color to the sequence.
- The game ends when you make a mistake.
